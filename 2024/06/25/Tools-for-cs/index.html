使用一些工具让自己更高效

工欲善其事必先利其器，hpc的代码编写往往是在linux环境下，很多IDE并不适配，因此掌握一些工具的使用变得极为必要

vim

幸运的是有chatgpt，我们获取知识的门槛大幅度降低，过一遍vim教程或许是必要的，但直接用vim写代码，遇到不会的去问gpt更加高效。
下面演示如何在ubuntu下安装vim，并配置一些常用插件，以实现代码自动补全等功能，并编写一个Hello World。

安装vim：

sudo apt update
sudo apt install vim
vim –version

然后可以输入vimtutor进行练习

经过简单的练习之后，可以配置自己的vim

用Vim-Plug管理插件

安装Vim-Plug
curl -fLo /.vim/autoload/plug.vim –create-dirs
https://raw.githubusercontent.com/junegunn/vim-plug/master/plug.vim
编辑
vim ~/.vimrc
我的是：
“ Vim-Plug 插件管理器初始化
call plug#begin(‘/.vim/plugged’)

“ 代码语法高亮和自动补全
Plug ‘dense-analysis/ale’
Plug ‘Valloric/YouCompleteMe’
Plug ‘scrooloose/nerdtree’
Plug ‘preservim/nerdcommenter’

“ 代码格式化
Plug ‘Chiel92/vim-autoformat’

“ 代码跳转和导航
Plug ‘majutsushi/tagbar’

“ 状态栏增强
Plug ‘vim-airline/vim-airline’
Plug ‘vim-airline/vim-airline-themes’

“ 其他有用的插件
Plug ‘tpope/vim-fugitive’ “ Git 集成

“ 初始化插件
call plug#end()

然后安装插件：
:PlugInstall

配置vimrc：

1.安装依赖
sudo apt install build-essential cmake vim-nox python3-dev
sudo apt install mono-complete golang nodejs default-jdk npm

2.编译 YouCompleteMe：
cd ~/.vim/plugged/YouCompleteMe
python3 install.py –all

3.ALE 提供实时的代码 linting 和静态分析，可以通过以下配置来启用：

“ 在 .vimrc 中添加
let g:ale_linters = {
\ ‘c’: [‘gcc’],
}

let g:ale_fixers = {
\ ‘c’: [‘clang-format’],
}

“ 启用 ALE 自动修复
let g:ale_fix_on_save = 1

4.NERDTree是一个文件系统浏览器，可以通过以下命令在 Vim 中打开：
:NERDTreeToggle

5.Tagbar提供代码的结构视图，可以通过以下命令在 Vim 中打开：
:TagbarToggle

  1. 示例 .vimrc 配置文件
    以下是一个示例 .vimrc 文件，包含上述插件的配置：

vim
set nocompatible “ 禁用兼容模式
filetype off “ 禁用文件类型检测

“ Vim-Plug 插件管理器初始化
call plug#begin(‘~/.vim/plugged’)

“ 插件列表
Plug ‘dense-analysis/ale’
Plug ‘Valloric/YouCompleteMe’
Plug ‘scrooloose/nerdtree’
Plug ‘preservim/nerdcommenter’
Plug ‘Chiel92/vim-autoformat’
Plug ‘majutsushi/tagbar’
Plug ‘vim-airline/vim-airline’
Plug ‘vim-airline/vim-airline-themes’
Plug ‘tpope/vim-fugitive’

“ 初始化插件
call plug#end()

“ 启用语法高亮
syntax on
filetype plugin indent on

“ 配置 ALE
let g:ale_linters = {
\ ‘c’: [‘gcc’],
}
let g:ale_fixers = {
\ ‘c’: [‘clang-format’],
}
let g:ale_fix_on_save = 1

“ 启用 NERDTree
map :NERDTreeToggle

“ 启用 Tagbar
map :TagbarToggle

“ 其他有用的配置
set number “ 显示行号
set expandtab “ 使用空格代替 Tab
set tabstop=4 “ Tab 宽度为 4
set shiftwidth=4 “ 缩进宽度为 4
set autoindent “ 自动缩进
set smartindent “ 智能缩进
保存并退出 .vimrc 文件后，重新打开 Vim 即可开始使用这些插件进行 C 语言的编写和开发。

gcc,g++,make
